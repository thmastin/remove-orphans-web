Collapse to Name, Image, and SKU (VLOOKUP)

This tool collapses all *_name columns into a single Name column, all *_image columns into a single Image column, and keeps all SKU columns (columns containing 'sku', case-insensitive). Empty rows are removed. Perfect for preparing a lookup table for Excel VLOOKUP or similar operations.
