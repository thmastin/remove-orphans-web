CSV Utility Tools

  • Keep Name, Image, and Import Columns
    Keeps all columns left of '1_name' and any columns matching patterns like *_name, *_image, or *_import. Useful for extracting key product info from large CSVs.
  • Keep Name and Image Columns
    Keeps only columns matching patterns like *_name and *_image. Use this to filter a CSV to just product names and images.
  • Keep Validation Columns
    Keeps only columns matching validation-related patterns. Use this to extract validation columns from a CSV.
  • Remove Empty Columns
    Removes columns that are empty, zero, or blank in all rows. Cleans up CSVs by dropping useless columns.
  • Remove Option UUIDs
    Clears the 'match_id', '1_uuid', and '2_uuid' columns for rows where type is 'opt'. Use this to anonymize or reset option UUIDs in your data.
  • Remove Orphaned Options from CSV
    Removes orphaned option rows from a CSV file. Upload your CSV, process it, and download both cleaned and orphaned rows.
